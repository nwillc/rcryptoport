# rcryptoport

A Rust version of my Go cryptoport application:

```shell
$ rcryptoport
         0.010345 BTC        328.64 (  -0.22%)
        07.030965 ETH       1274.04 (  -0.38%)
               Total:       1602.69 (  -0.35%)
```

## Configuration
Have your Nomics API key and holdings info ready. Setup your configuration:

```shell
$ rcryptoport setup
```
